# SSMK 점수 모델 v1

작성일: 2026-04-22

## 1. 모델의 목적

SSMK 점수 모델은 매수/매도 결정을 대신하기 위한 모델이 아니다. 이 모델은 AI와 초보자가 함께 미국주식을 관찰하고, 지표의 의미를 배우고, 가설을 세운 뒤 실제 결과와 비교하기 위한 학습형 점수 모델이다.

핵심 원칙:

```text
점수는 정답이 아니다.
점수는 이번 주에 어떤 질문을 먼저 던질지 정하는 도구다.
```

모든 주요 판단은 아래 구조로 기록한다.

```text
가설 요약
→ 근거 지표/데이터
→ 이 지표/데이터를 보고 왜 그렇게 예상했나
→ 초보자가 배울 포인트
→ 한계와 다음 확인
```

---

## 2. 점수 표현 방식

SSMK는 10점 만점 숫자 점수를 유지한다. 다만 숫자만 쓰면 초보자가 정답처럼 받아들일 수 있으므로, `관찰 등급`과 `데이터 신뢰도`를 함께 표시한다.

| 항목 | 의미 | 예시 |
|---|---|---|
| SSMK 점수 | 10점 만점의 관찰 점수 | `8.4` |
| 관찰 등급 | 점수를 쉬운 말로 바꾼 표현 | `높음` |
| 데이터 신뢰도 | 사용한 자료를 얼마나 믿을 수 있는지 | `중간` |
| 불확실성 | 아직 모르는 것이 얼마나 큰지 | `높음` |

관찰 등급 기준:

| SSMK 점수 | 관찰 등급 | 의미 |
|---:|---|---|
| 8.0~10.0 | 높음 | 이번 주에 먼저 공부할 이유가 큼 |
| 6.0~7.9 | 중간 | 계속 지켜보며 변화 이유를 확인 |
| 0.0~5.9 | 낮음 | 우선순위는 낮지만 리스크나 회복 가능성은 기록 |

주의:

```text
관찰 등급 높음 = 좋은 투자 대상이라는 뜻이 아니다.
관찰 등급 높음 = 이번 주에 더 먼저 공부할 이유가 있다는 뜻이다.
```

## 3. SSMK 총점 구조

기본 점수는 6개 항목으로 구성한다.

| 항목 | 비중 | 의미 |
|---|---:|---|
| 기업 체력 점수 | 30% | 매출, 이익, EPS, 현금흐름, 부채 |
| 배당/주주환원 점수 | 20% | 배당성장, 배당성향, 자사주매입 |
| 산업 매력도 점수 | 20% | 산업 성장률, 침투율, 시장점유율, 구조적 트렌드 |
| 비즈니스 모델 점수 | 15% | 매출 구성, 반복매출, 마진, 경쟁력 |
| 가격/타이밍 점수 | 10% | 밸류에이션, 주가 위치, 배당률 위치 |
| 내부자/이벤트 점수 | 5% | 내부자 매매, 실적 발표, 제품 출시, 규제 이벤트 |

계산식:

```text
SSMK 총점 =
기업 체력 점수 * 0.30
+ 배당/주주환원 점수 * 0.20
+ 산업 매력도 점수 * 0.20
+ 비즈니스 모델 점수 * 0.15
+ 가격/타이밍 점수 * 0.10
+ 내부자/이벤트 점수 * 0.05
```

## 4. 투자 성격별 보정

모든 기업을 같은 기준으로 보면 왜곡이 생길 수 있다. 예를 들어 NVDA 같은 성장주와 JNJ 같은 배당주는 봐야 할 포인트가 다르다.

초기 v1에서는 기본 비중을 유지하되, 해석에서 투자 성격을 반드시 함께 적는다.

| 투자 성격 | 더 중요하게 볼 항목 | 이유 |
|---|---|---|
| 성장주 | 매출 성장, EPS 성장, 산업 매력도, 밸류에이션 부담 | 성장 기대가 실제 실적으로 이어지는지 봐야 함 |
| 배당주 | 현금흐름, 배당성향, 배당 성장, 부채 | 배당이 지속 가능한지 봐야 함 |
| 방어주 | 이익 안정성, 현금흐름, 경기 민감도 | 시장이 흔들릴 때 버티는 힘을 봐야 함 |
| 턴어라운드 | 매출 회복, 마진 개선, 부채, 비용 절감 | 좋아지는 초입인지 일시적 반등인지 봐야 함 |
| 경기민감주 | 원자재 가격, 수요 사이클, 마진, 재고 | 경기와 가격 사이클 영향을 크게 받음 |

향후 v2에서는 투자 성격별 가중치를 다르게 적용할 수 있다.

## 5. 데이터 신뢰도 기준

데이터 신뢰도는 점수보다 먼저 확인해야 하는 안전장치다.

| 신뢰도 | 기준 | 예시 |
|---|---|---|
| 높음 | 공식 원자료가 있고 최신성이 확인됨 | SEC 공시, 기업 IR, 공식 배당 발표 |
| 중간 | 공식 자료 일부와 보조 출처가 함께 있음 | 기업 IR + 뉴스, 데이터 사이트 + 기업 발표 |
| 낮음 | 단일 뉴스, 추정치, 자동 수집 오류 가능성이 있음 | 단일 기사, 블로그, 확인되지 않은 일정 |

중요:

```text
SSMK 점수가 높아도 데이터 신뢰도가 낮으면 가설 확신도를 낮춘다.
데이터 신뢰도가 낮은 항목은 리포트에 반드시 한계로 표시한다.
```

## 6. 불확실성 기준

불확실성은 "AI가 얼마나 조심해야 하는가"를 표시한다.

| 불확실성 | 기준 | 예시 |
|---|---|---|
| 낮음 | 공식 수치가 있고 해석이 비교적 명확함 | 배당 발표, 확정 실적 수치 |
| 중간 | 데이터는 있지만 해석이 여러 방향으로 가능함 | 매출은 증가했지만 마진이 감소 |
| 높음 | 기대, 뉴스, 추정이 많고 실제 확인 전임 | AI 제품 기대, 미확정 실적 일정 |

## 7. 가설 작성 규칙

모든 중요한 리포트 문장은 아래 5단 구조를 따른다.

| 구성 | 설명 |
|---|---|
| 가설 요약 | 이번 데이터로 예상해 볼 수 있는 관찰 가설 |
| 근거 지표/데이터 | 어떤 수치, 뉴스, 이벤트를 봤는지 |
| 해석 이유 | 그 데이터가 왜 그런 가설로 이어지는지 |
| 초보자 레슨 | 이 지표가 무엇을 보여주는지 쉬운 말로 설명 |
| 한계와 다음 확인 | 무엇이 아직 불확실하고 무엇을 나중에 봐야 하는지 |

예시:

| 구성 | 내용 |
|---|---|
| 가설 요약 | 배당률 상승이 매력 증가가 아니라 고배당 착시일 수 있다. |
| 근거 지표/데이터 | 배당금은 그대로인데 주가가 하락해 배당률이 상승했다. |
| 해석 이유 | 배당률은 배당금/주가이므로 주가가 떨어져도 높아진다. |
| 초보자 레슨 | 배당률 상승은 좋은 소식일 수도 있고, 주가 하락의 결과일 수도 있다. |
| 한계와 다음 확인 | 다음 실적에서 이익과 현금흐름이 배당을 감당하는지 확인해야 한다. |

## 8. 항목별 점수 기준

### 8-1. 기업 체력 점수

| 점수 구간 | 기준 |
|---|---|
| 8~10 | 매출, 이익, EPS, 현금흐름이 대체로 함께 개선 |
| 6~7 | 주요 지표는 양호하지만 일부 둔화나 확인 필요 |
| 0~5 | 매출/이익/현금흐름 중 여러 항목에 약점 |

초보자 레슨:

```text
기업 체력은 회사가 실제로 돈을 잘 벌고 버틸 수 있는지 보는 항목입니다.
매출만 늘고 이익이 줄면, 많이 팔지만 남는 돈이 줄어드는 상황일 수 있습니다.
```

### 8-2. 배당/주주환원 점수

| 점수 구간 | 기준 |
|---|---|
| 8~10 | 배당/자사주매입이 지속 가능하고 현금흐름이 뒷받침 |
| 6~7 | 주주환원은 있으나 성장성이나 지속성 확인 필요 |
| 0~5 | 배당 부담, 삭감 위험, 주주환원 부족 |

초보자 레슨:

```text
배당률이 높다고 무조건 좋은 것은 아닙니다.
배당금이 늘어서 높아진 것인지, 주가가 떨어져서 높아진 것인지 구분해야 합니다.
```

### 8-3. 산업 매력도 점수

| 점수 구간 | 기준 |
|---|---|
| 8~10 | 구조적 성장 흐름이 있고 대표 지표가 개선 |
| 6~7 | 산업은 유망하지만 사이클/경쟁/정책 확인 필요 |
| 0~5 | 수요 둔화, 경쟁 심화, 규제 리스크가 큼 |

초보자 레슨:

```text
산업은 회사가 장사하는 상권입니다.
좋은 회사도 상권이 나빠지면 성장하기 어렵고, 상권이 커지면 기회가 늘어날 수 있습니다.
```

### 8-4. 비즈니스 모델 점수

| 점수 구간 | 기준 |
|---|---|
| 8~10 | 반복매출, 높은 마진, 강한 경쟁우위가 있음 |
| 6~7 | 사업 구조는 양호하지만 특정 제품/고객 의존 확인 필요 |
| 0~5 | 매출 구조가 불안정하거나 경쟁우위가 약함 |

초보자 레슨:

```text
비즈니스 모델은 회사가 돈을 버는 방식입니다.
한 번 팔고 끝나는 사업인지, 매달 반복적으로 돈이 들어오는 사업인지가 중요합니다.
```

### 8-5. 가격/타이밍 점수

| 점수 구간 | 기준 |
|---|---|
| 8~10 | 역사적 밸류에이션이나 주가 위치상 재점검 가치가 큼 |
| 6~7 | 과열도 침체도 아닌 중립 구간 |
| 0~5 | 기대가 많이 반영됐거나 가격 부담이 큼 |

초보자 레슨:

```text
좋은 회사도 너무 비싸게 사면 좋은 결과가 나오기 어렵습니다.
가격/타이밍 점수는 사라는 뜻이 아니라, 현재 기대가 가격에 얼마나 반영됐는지 보자는 뜻입니다.
```

### 8-6. 내부자/이벤트 점수

| 점수 구간 | 기준 |
|---|---|
| 8~10 | 의미 있는 내부자 매수, 실적/제품/규제 이벤트가 긍정적으로 확인 |
| 6~7 | 이벤트는 있으나 영향 확인 필요 |
| 0~5 | 부정 이벤트, 불확실성, 내부자 매도 해석 주의 |

초보자 레슨:

```text
이벤트는 회사의 방향이 바뀔 수 있는 일정입니다.
하지만 이벤트 하나만으로 판단하지 않고, 실제 실적과 지표에 반영되는지 확인해야 합니다.
```

## 9. 리스크 플래그 기준

| 리스크 플래그 | 의미 |
|---|---|
| valuation_risk | 가격 부담 또는 기대 과열 가능성 |
| dividend_trap_risk | 고배당 착시 가능성 |
| margin_risk | 이익률 하락 가능성 |
| growth_slowdown_risk | 성장 둔화 가능성 |
| regulation_risk | 규제/소송/정책 리스크 |
| data_quality_risk | 데이터 신뢰도 낮음 |
| event_uncertainty_risk | 이벤트 결과 불확실 |

## 10. 관찰 후보 3개 선정 기준

관찰 후보 3개는 좋은 종목 순위가 아니다. 이번 주에 학습 가치가 큰 후보를 고르는 것이다.

선정 기준:

1. SSMK 점수 또는 점수 변화가 크다.
2. 가설을 세울 만한 지표 변화가 있다.
3. 초보자가 배울 수 있는 지표 해석 포인트가 있다.
4. 리스크나 한계가 분명해서 복기 가치가 있다.
5. 데이터 신뢰도가 너무 낮으면 후보로 올리되 `재확인 필요`를 표시한다.

| 근거 지표/데이터 | 클라우드 성장률, AI 관련 투자, 영업이익률 |

| 초보자 레슨 | 기대감은 뉴스에 먼저 나타나지만, 진짜 힘은 매출과 이익에서 확인된다. |
| 한계 | 최신 실적 발표 전이면 실제 반영 여부는 불확실하다. |
| 다음 확인 | 다음 실적에서 클라우드 성장률과 AI 비용을 확인한다. |
